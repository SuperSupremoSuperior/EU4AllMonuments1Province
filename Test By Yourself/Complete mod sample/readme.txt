To create your owm mod, you must:

1 - Go to the game laucher

2 - Go to "All installed mods"

3 - "UPLOAD MOD" (on top right)

4 - "CREATE A MOD"

5 - Use * for any version in the game, and chose the name and the type of the mod in this case de type don't matter

6 - The mod folder is created with the correct descriptor.mod

NOTE: The mod file will be founded in the documents, or you can find it by puting the create mod in a Playset...
