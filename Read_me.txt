--- En-Us ---
This program is very simple to use:

1 - Run the program.

2 - Enter a value for the province.
Note: Values can be found on the game's wiki or by using "debug_mode" and hovering the mouse over the province in-game.

3 - Enter a value between 0 and 3 to define the tier.
Note: I left it unlimited because some mods might include hardcoded features that allow for more levels.

4 - Select the folder containing the .txt files for monuments.

Important: Be careful, as the program will replace all files in the selected folder.
I recommend creating a separate folder, copying the files you want to modify into it, and working from there. This ensures you won’t accidentally break anything important in the game or your mod.

The application is currently limited to these functions. More features may be added in the future.
